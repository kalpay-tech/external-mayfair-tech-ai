# Success Stories Images

This folder contains images for the "Success Stories" section on the homepage.

## Required Images

Place the following images in this folder:

1. **manufacturing.jpg** - Smart Manufacturing AI
   - Suggested content: Factory floor, robotic automation, quality control systems
   - Aspect ratio: 16:10 (e.g., 1600x1000px)

2. **healthcare.jpg** - Predictive Healthcare Platform
   - Suggested content: Medical technology, healthcare professionals, diagnostic systems
   - Aspect ratio: 16:10 (e.g., 1600x1000px)

3. **logistics.jpg** - Intelligent Supply Chain
   - Suggested content: Warehouse, automated logistics, inventory management
   - Aspect ratio: 16:10 (e.g., 1600x1000px)

## Image Guidelines

- **Format**: JPG or PNG (JPG recommended for photos)
- **Dimensions**: Minimum 1600x1000px (16:10 aspect ratio)
- **File Size**: Optimize for web (under 500KB per image)
- **Style**: Professional, high-quality, relevant to each industry
- **Colors**: Should complement the MayfairTech color scheme

## How It Works

Simply add your images with the exact filenames above, and they'll automatically appear in the Success Stories section with:
- Category badges overlay
- Hover zoom effects
- Responsive design
- Automatic fallback if images are missing

The layout matches your reference design with images at the top of each card!
