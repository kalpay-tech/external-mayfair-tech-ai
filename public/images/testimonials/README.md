# Testimonial Images

This folder contains profile images for the "What Our Clients Say" section on the homepage.

## Required Images

Place the following images in this folder:

1. **sarah.jpg** - Sarah Johnson, CTO of TechCorp Inc.
2. **michael.jpg** - Michael Rodriguez, CEO of FastShip Logistics
3. **emily.jpg** - Dr. Emily Watson, Director of Operations at HealthFirst

## Image Guidelines

- **Format**: JPG or PNG (JPG preferred for photos)
- **Dimensions**: 200x200px minimum (square aspect ratio 1:1)
- **File Size**: Optimize for web (under 200KB per image)
- **Style**: Professional headshots or portraits
- **Background**: Preferably solid or simple backgrounds

## How It Works

The testimonial carousel will automatically display these images with:
- Circular crop
- White border
- Company badge overlay
- Fallback to initials if image is missing

Simply add profile images with the exact filenames above, and they'll appear in the testimonial slider!

## Fallback Behavior

If images are not added, the component will show:
- User initials in a colorful gradient circle
- Company badge overlay
- No errors or broken images
